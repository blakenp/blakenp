## Current Projects
Below are some of the most interesting projects I am currently working on. This isn't all I am up to, but these are the ones I find most interesting. You can check out more of my projects on my website at [blakenp.com](https://blakenp.com/).
### Codenames
I am currently working as a researcher at BYU on a project helping AI agents learn to make intelligent decisions in the language game *Codenames*. We use vector embeddings and cosine distance to demonstrate AI's effectiveness in this game, and the bot Spymaster leverages learned embeddings to generate clues, guiding both human and bot players in identifying their team's spies.
### Competitive Programming Tutor
Another project I am currently working on is a website that will use an LLM backed up with RAG to answer questions about competitive programming topics. The idea is to have a good tutor for preparing for coding competitions and technical interviews. I think this is really important since a lot of people look for more ways to practice, and I really enjoy competitive programming and learning and understanding more about algorithms used in coding competitions. I'm super excited about this one and am having fun with it.
